# J.A.R.V.I.S. Landing Page

Esta es la landing page oficial para la extensión de voz de J.A.R.V.I.S. (Gemini Voice CLI Extension). 

## Estética y Diseño
- **Tema:** Dark Mode (Inspirado en interfaces tecnológicas y J.A.R.V.I.S.).
- **Colores:** Indigo, Violeta y Esmeralda para acentos.
- **Tipografía:** Inter para texto y Fira Code para bloques de código.
- **Efectos:** Gradientes radiales y animaciones suaves de entrada (fade-in).

## Estructura de Archivos
- `src/App.tsx`: Componente principal con el Hero Section y Footer.
- `src/components/ProjectInfo.tsx`: Cuerpo detallado con la información técnica del proyecto.
- `src/styles/App.css`: Estilos globales y específicos de la landing page.
- `src/main.tsx`: Punto de entrada de React.
- `index.html`: Estructura base HTML.

## Cómo Ejecutar (Desarrollo)

Para visualizar esta landing page localmente, puedes usar herramientas como **Vite** o **Parcel**.

### Opción 1: Usando Vite (Recomendado)
Si tienes Node.js instalado, puedes ejecutar lo siguiente desde la raíz de `apps/jarvis-landing/`:

```bash
# Instalar dependencias necesarias (si usas un gestor de paquetes)
npm install react react-dom
npm install -D @types/react @types/react-dom vite

# Ejecutar el servidor de desarrollo
npx vite
```

### Opción 2: Integración Estática
Los archivos `.tsx` y `.css` están diseñados para ser modulares. Puedes copiar la carpeta `src` en cualquier proyecto existente de React para integrarla rápidamente.
